Tambahkan sistem baru bernama "📊 DATABASE" tanpa mengubah fitur lama. Semua fitur lama harus tetap berjalan normal. Gunakan Discord.js v14, kode modular, rapi, mudah dikembangkan, dan semua tampilan menggunakan Bahasa Indonesia.

==================================================
📊 KATEGORI DATABASE
==================================================

Kategori Database berisi:

⚙️・bot-setting
📦・backup
📄・console
👥・member-list

==================================================
/SETUP
==================================================

Tambahkan command:

/setup

Saat dijalankan muncul menu admin (ephemeral).

Pilihan:

📊 Database
❌ Tutup

Jika memilih Database dan belum pernah setup:

- Minta admin memilih channel:
  • Bot Setting
  • Backup
  • Console
  • Member List

Setelah selesai tampilkan ringkasan lalu tombol:

✅ Buat Panel
✏️ Edit
❌ Batal

Jika memilih "Buat Panel", bot otomatis mengirim panel ke channel yang dipilih.

Jika Database sudah pernah dibuat, jangan setup lagi.

Tampilkan menu:

📝 Edit Setup
🔄 Buat Ulang Panel
🗑 Hapus Panel
❌ Tutup

"Hapus Panel" hanya menghapus panel yang dibuat bot.

JANGAN menghapus:
- Channel Discord
- Database
- Backup GitHub
- Riwayat Console

==================================================
💾 BACKUP
==================================================

Saat Backup ditekan:

Bot membuat backup ZIP.

Backup harus bisa:

✔ Download langsung dari Discord.

✔ Upload ke GitHub.

Backup berisi:

- Database
- Config
- Settings
- User Data
- Premium Data
- BoomBox
- Scanner
- AI Data
- Plugin Data
- Assets
- Logs
- Session
- Custom Command
- backup-info.json

JANGAN backup:

- node_modules
- npm cache
- cache sementara
- temp
- folder kosong
- file sampah

Backup selesai menampilkan tombol:

📥 Download
☁ Upload GitHub

==================================================
DETAIL
==================================================

Setiap file wajib memiliki alasan.

Contoh:

cache/
Alasan:
Cache lama.

temp/
Alasan:
File sementara.

logs/
Alasan:
Log lebih dari 30 hari.

backup/
Alasan:
Backup lama.

music.mp3

Alasan:
Tidak dipakai bot.
Tidak ada di database.
Tidak pernah diakses 90 hari.

Rekomendasi:
Periksa sebelum dihapus.

config.json

File sistem.
Tidak boleh dihapus.

ATURAN
