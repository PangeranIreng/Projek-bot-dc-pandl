TARGET

Artinya:
yt-dlp

upload Top4Top
premium
limit
downloader
cache
database
tetap dipakai.
Yang dirubah hanyalah struktur dan UI.
1. Setup BoomBox
Hapus sistem setup BoomBox lama.
Ganti menjadi SATU command:
/setupboombox
Panel utama harus berisi:
📺 Setup Channel

📋 Setup BoomBox Logs

⏱️ Batas Durasi

🛠️ Maintenance
Tidak boleh membuat command setup lain.
Semua konfigurasi BoomBox harus melalui panel ini.
2. Setup Channel
Support:
YouTube

TikTok

Spotify
Owner memilih platform.
Lalu memilih channel Discord.
Semua channel disimpan ke database.
Tidak boleh hardcode Channel ID.
3. Setup BoomBox Logs
BoomBox Logs hanya menggunakan SATU channel.
Misalnya:
#boombox-logs
Bukan tiga channel.
Semua logs dikirim ke channel tersebut.
Tetapi di dalam panel Logs dipisahkan menjadi:
📺 Logs BoomBox YouTube

🎵 Logs BoomBox TikTok

🎧 Logs BoomBox Spotify
4. BoomBox Logs
JANGAN HAPUS fitur BoomBox Logs lama.
Yang diubah hanyalah UI dan struktur.
Semua log YouTube lama harus tetap ada.
Semua log lama dipindahkan menjadi:
Logs BoomBox YouTube
TikTok dan Spotify mulai kosong.
5. Tampilan BoomBox Logs
Saat membuka BoomBox Logs.
Pertama tampil:
Pilih Platform

📺 YouTube

🎵 TikTok

🎧 Spotify

Tampilkan maksimal:
5 log
per halaman.
Di bawah embed harus ada:
Buttons
First

Prev

Refresh

Next

Last
Lalu Dropdown
Pilih Halaman
Lalu Dropdown
Ganti Platform
Jangan menghapus tombol lama.
Rapikan saja.
6. Penyimpanan Logs
Saat user berhasil convert.
Bot tetap membalas URL BoomBox ke user.
Setelah itu.
Bot otomatis menyimpan:
Platform
Judul
URL BoomBox
Tanggal
Jam
ke BoomBox Logs.
YouTube masuk ke:
Logs BoomBox YouTube
TikTok masuk ke:
Logs BoomBox TikTok
Spotify masuk ke:
Logs BoomBox Spotify
BoomBox Logs berfungsi sebagai arsip URL.
Bukan hanya history.
7. Batas Durasi
Jangan hardcode role.
Bot harus membaca semua Role Discord secara otomatis.
Semua role server harus muncul.
Jika server membuat role baru.
Bot otomatis mendeteksi.
Owner tinggal memilih role.
Lalu mengatur durasi maksimal.
Durasi disimpan ke database.
8. Maintenance
Maintenance harus berada di:
/setupboombox
Support:
YouTube

TikTok

Spotify

Semua Platform

🚧 BoomBox sedang maintenance.

TikTok dan Spotify tetap berjalan.
