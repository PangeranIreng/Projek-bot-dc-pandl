==========================================================
TARGET
==========================================================

BoomBox harus terasa seperti bot profesional.

Cepat.

Rapi.

Tidak spam.

Tidak bikin channel penuh.

Tidak bikin user bingung.

==========================================================
1. REPLY USER
==========================================================

Saat user mengirim link BoomBox.

Bot WAJIB reply ke pesan user.

Bukan kirim pesan baru.

Contoh:

User:

https://youtu....

↓

Bot reply:

@User

🎵 Sedang membuat BoomBox...

==========================================================
2. LOADING REALISTIS
==========================================================

Gunakan hanya SATU embed.

Jangan spam banyak pesan.

Embed tersebut di-edit terus.

Gunakan spinner unicode.

Contoh:

⠋
⠙
⠹
⠸
⠼
⠴
⠦
⠧
⠇
⠏

Spinner berubah setiap edit.

==========================================================
3. TAHAP LOADING
==========================================================

Contoh:

⠋ Menghubungkan...

↓

⠹ Mengambil Metadata...

↓

⠼ Menyiapkan Audio...

↓

⠦ Upload BoomBox...

↓

⠇ Verifikasi Link...

↓

Selesai.

Jangan menambahkan delay.

Jika tahap selesai lebih cepat langsung pindah.

==========================================================
4. HAPUS EMBED PROSES
==========================================================

Saat BoomBox berhasil.

Embed Processing langsung DIHAPUS.

Setelah dihapus.

Bot mengirim embed baru.

Contoh:

@User

━━━━━━━━━━━━━━━━━━

✅ BoomBox Berhasil

🎵 Judul

⏱ Durasi

🔗 BoomBox URL

━━━━━━━━━━━━━━━━━━

Powered by PangeranAsisten | BoomBox

Jadi channel bersih.

Tidak ada embed loading yang tertinggal.

==========================================================
5. ERROR USER
==========================================================

Jika gagal.

Embed Processing dihapus.

Bot reply user:

@User

⚠️ BoomBox sedang mengalami gangguan.

Laporan otomatis telah dikirim ke Owner & Developer.

Silakan coba beberapa saat lagi.

Powered by PangeranAsisten | BoomBox

Jangan tampilkan:

Stack Trace

Provider

API

Internal Error

==========================================================
7. SUPPORT TIKTOK
==========================================================

Perbaiki parser TikTok.

Support seluruh format:

https://tiktok.com/

https://www.tiktok.com/

https://m.tiktok.com/

https://vt.tiktok.com/

https://vm.tiktok.com/

Short Link

Share Link terbaru

Pastikan parser melakukan resolve redirect.

Jangan sampai link valid dianggap Unsupported URL.

==========================================================
8. VALIDASI URL
==========================================================

Jika platform belum didukung.

Reply user:

⚠️ Platform belum didukung.

Saat ini BoomBox hanya mendukung:

• YouTube

• YouTube Music

• TikTok

Powered by PangeranAsisten | BoomBox

==========================================================
9. PERFORMANCE
==========================================================

Optimalkan seluruh proses.

Target:

Metadata

≤2 detik

Download

≤5 detik

BoomBox

≤8 detik

Kurangi seluruh delay yang tidak perlu.

==========================================================
10. TIMEOUT
==========================================================

Provider timeout maksimal:

5-8 detik.

Jika timeout.

Langsung provider berikutnya.

==========================================================
11. ANTI STUCK
==========================================================

Cari seluruh kemungkinan:

Promise menggantung.

Worker menggantung.

Queue Lock.

Memory Leak.

Zombie Stream.

Perbaiki.

Pastikan finally selalu release Queue.
